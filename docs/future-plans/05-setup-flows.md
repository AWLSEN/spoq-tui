# Setup Flows and User Experience

## Overview

This document details the user experience flows for both Cloud and Jetson products, focusing on making setup as simple as possible while maintaining security.

## Design Principles

1. **Zero technical knowledge required** - No mention of SSH, WireGuard, ports, etc.
2. **Familiar patterns** - Like setting up AirPods or a new iPhone
3. **Progressive disclosure** - Advanced options hidden, simple path prominent
4. **Recovery built-in** - Easy to restart setup if something goes wrong
5. **Instant gratification** - Working system within 5 minutes
